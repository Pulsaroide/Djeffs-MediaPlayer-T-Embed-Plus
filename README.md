# 🎬 Djeff's MediaPlayer For T-Embed Plus

A full-featured video & audio media player for the **LilyGo T-Embed CC1101 Plus** (ESP32-S3).

![Framework](https://img.shields.io/badge/Framework-Arduino-green)

## ✨ Features

| Feature | Details |
|---------|---------|
| 🎥 Video playback | MJPEG/AVI @ 170×270, up to 30fps |
| 🔊 Audio | PCM stereo 44100Hz via I2S speaker |
| 📂 File browser | Scroll through `/videos/` on SD card |
| 🖼️ Thumbnails | Auto-loads from `/thumbs/` (JPEG 48×30) |
| ▶️ Play/Pause/Stop | Encoder press = Pause, long press = Back |
| 📊 Progress bar | Real-time position + duration |
| 🔈 Volume control | Encoder rotation while playing |
| 🔁 Loop | Toggle in browser header |

## 🚀 Getting Started

### 1. Clone this repo
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
```

### 2. Build firmware (GitHub Actions — no tools needed!)
- Push to `main` → firmware auto-builds
- Download `.bin` from **Releases** or **Actions → Artifacts**

### 3. Flash the firmware
```bash
pip install esptool

esptool.py --chip esp32s3 --baud 921600 \
  write_flash --flash_mode dio --flash_size 16MB \
  0x0      bootloader.bin \
  0x8000   partitions.bin \
  0xe000   boot_app0.bin \
  0x10000  Djeffs-MediaPlayer-T-Embed-Plus.bin
```
> Hold **BOOT** button, press **RESET**, then run the command.

## 🎬 Preparing Videos

### Recommended format
| Parameter | Value |
|-----------|-------|
| Container | AVI |
| Video codec | MJPEG |
| Resolution | 170×270 (auto-padded) |
| Frame rate | 25fps |
| MJPEG quality | 5 (1=best, 31=worst) |
| Audio codec | PCM 16-bit signed LE |
| Sample rate | 44100 Hz |
| Channels | Stereo |

### Convert with FFmpeg

**Linux/Mac:**
```bash
chmod +x tools/convert_videos.sh
./tools/convert_videos.sh input.mp4           # single file
./tools/convert_videos.sh /path/to/videos/   # whole folder
```

**Windows:**  
Drag your video file onto `tools/convert_videos.bat`

**Manual FFmpeg command:**
```bash
ffmpeg -i input.mp4 \
  -vf "scale=170:270:force_original_aspect_ratio=decrease,pad=170:270:(ow-iw)/2:(oh-ih)/2:black,fps=25" \
  -vcodec mjpeg -q:v 5 \
  -acodec pcm_s16le -ar 44100 -ac 2 \
  -f avi output.avi
```

### SD Card Structure
```
SD Card (32GB)
├── videos/
│   ├── movie1.avi
│   ├── movie2.avi
│   └── ...
└── thumbs/
    ├── movie1.jpg   ← 48×30 JPEG thumbnail (optional)
    ├── movie2.jpg
    └── ...
```
> Thumbnails are auto-generated by the convert script.

|--------|--------|
